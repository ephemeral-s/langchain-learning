# LangChain Learning

LangChain 学习笔记与代码实践。

## 项目结构

```
langchain-learning/
├── basics/          # 基础概念与入门示例
├── prompts/         # Prompt 模板与管理
├── chains/          # 链式调用
├── agents/          # 智能代理
├── memory/          # 记忆与上下文管理
├── rag/             # 检索增强生成 (RAG)
├── tools/           # 工具调用
└── projects/        # 综合项目练习
```

## 环境配置

```bash
python -m venv venv
source venv/bin/activate
pip install langchain langchain-openai langchain-community
```
